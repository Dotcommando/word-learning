# Λέξεις

Однофайловое браузерное приложение для изучения греческих слов и их склонений.

Пользователь загружает собственный набор слов в формате JSON, пытается вспомнить перевод, открывает отдельные переводы по мере необходимости и просматривает формы слова в единственном и множественном числе.

## Возможности

- загрузка пользовательского набора слов из JSON-файла;
- таблица со словом, транскрипцией, переводом и примером употребления;
- скрытые по умолчанию переводы в виде мягких размытых плашек;
- независимое открытие перевода слова и перевода примера;
- единый переключатель показа всех переводов;
- склонения по четырём падежам в единственном и множественном числе;
- одновременное раскрытие склонений нескольких слов;
- светлая и тёмная темы;
- сохранение состояния интерфейса между перезагрузками;
- сохранение загруженных наборов слов в IndexedDB;
- восстановление последнего активного набора после перезагрузки;
- работа без сервера после сборки: приложение помещается в один HTML-файл.

## Быстрый старт

### Готовая сборка

Откройте `dist/index.html` в современном браузере.

Для полноценной проверки поведения IndexedDB предпочтительно запускать файл через локальный HTTP-сервер:

```bash
npx serve dist
```

После запуска откройте адрес, показанный командой.

### Разработка

```bash
npm install
npm run dev
```

Проверки и production-сборка:

```bash
npm run check
npm run build
```

Результат сборки:

```text
dist/index.html
```

CSS, JavaScript и SVG-иконки встраиваются непосредственно в этот файл.

## Как загрузить набор слов

1. Нажмите кнопку с иконкой загрузки.
2. Выберите файл с расширением `.json`.
3. Приложение проверит структуру файла.
4. При успешной проверке набор будет сохранён в IndexedDB и станет активным.
5. После перезагрузки страницы последний активный набор будет восстановлен автоматически.

Если JSON содержит ошибку, приложение покажет путь к проблемному полю и не сохранит частично разобранный набор.

## Формат JSON

Корневой объект описывает один набор слов.

```text
word set
├── id                  необязательный идентификатор набора
├── name                название набора
└── words               непустой массив слов
```

### Поля набора

| Поле | Тип | Обязательно | Описание |
|---|---|---:|---|
| `id` | `string` | нет | Стабильный идентификатор набора. Если отсутствует, приложение создаст его автоматически. |
| `name` | `string` | да | Отображаемое название набора. Строка не должна быть пустой. |
| `words` | `array` | да | Непустой массив объектов слов. |

### Поля слова

| Поле | Тип | Обязательно | Описание |
|---|---|---:|---|
| `id` | `string` | нет | Уникальный идентификатор слова внутри набора. Если отсутствует, создаётся автоматически. |
| `word` | `string` | да | Греческое слово в основной форме. |
| `transcription` | `string` | да | Транскрипция или подсказка по произношению. |
| `translation` | `string` | да | Перевод слова на русский язык. |
| `example` | `string` | да | Пример употребления на греческом. |
| `exampleTranslation` | `string` | да | Перевод примера на русский язык. |
| `declensions` | `object` | да | Формы единственного и множественного числа. |

Все строковые поля должны содержать непустой текст.

### Объект `declensions`

Объект должен содержать два массива:

| Поле | Тип | Содержимое |
|---|---|---|
| `singular` | `array` | Четыре падежные формы единственного числа. |
| `plural` | `array` | Четыре падежные формы множественного числа. |

В каждом массиве должны быть ровно четыре записи в следующем порядке:

1. `nominative` — именительный;
2. `genitive` — родительный;
3. `accusative` — винительный;
4. `vocative` — звательный.

Другой порядок считается ошибкой импорта.

### Поля формы склонения

| Поле | Тип | Обязательно | Описание |
|---|---|---:|---|
| `case` | `string` | да | Одно из четырёх допустимых значений падежа. |
| `form` | `string` | да | Греческая форма слова. Артикль можно включать в значение. |
| `translation` | `string` | да | Русский перевод или краткое пояснение формы. |
| `example` | `string` | да | Пример употребления формы на греческом. |
| `exampleTranslation` | `string` | да | Перевод примера на русский язык. |

Допустимые значения `case`:

```text
nominative
genitive
accusative
vocative
```

```json
            "exampleTranslation": "Матери, идите с нами."
          }
        ]
      }
    }
  ]
}
```

## Проверка импортируемого файла

Импорт будет отклонён, если:

- корневое значение не является объектом;
- отсутствует `name`;
- `words` не является непустым массивом;
- обязательное строковое поле отсутствует или пусто;
- `id` указан не строкой;
- два слова имеют одинаковый `id`;
- отсутствует `singular` или `plural`;
- массив склонений содержит не четыре записи;
- падежи расположены не в каноническом порядке;
- падеж повторяется;
- значение `case` не входит в список допустимых;
- файл содержит синтаксически некорректный JSON.

Проверка выполняется до записи в IndexedDB. При ошибке новый набор не сохраняется.

## Хранение данных

Приложение использует два браузерных хранилища.

### localStorage

В `localStorage` сохраняется только состояние интерфейса:

- выбранная тема;
- глобальная видимость переводов;
- отдельно открытые переводы;
- раскрытые строки склонений;
- идентификатор активного набора.

Ключ текущей версии:

```text
word-learning/ui-state
```

### IndexedDB

Содержимое загруженных наборов сохраняется в IndexedDB.

```text
Database: word-learning-word-sets
Object store: wordSets
```

Сейчас интерфейс восстанавливает последний активный набор. Выбор из всех ранее загруженных наборов планируется как отдельная функция.

## Работа без сети

Основная логика приложения, стили и иконки находятся внутри итогового HTML-файла. После первой загрузки набора его данные хранятся локально в браузере.

Приложение не отправляет загруженные слова на удалённый сервер.

## Ограничения текущей версии

- поддерживается только JSON;
- импортированный набор нельзя редактировать в интерфейсе;
- пока нет списка ранее загруженных наборов;
- нет экспорта и удаления наборов;
- склонения должны быть подготовлены заранее;
- приложение не генерирует грамматические формы автоматически.
